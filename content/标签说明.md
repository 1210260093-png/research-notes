# 标签说明

这篇笔记维护当前 vault 的核心标签。标签分为三类：状态、对象、方法。日常写笔记时优先使用下面这些标签，避免为同一概念发明多个近义标签。

## 状态标签

| 标签 | 含义 |
|---|---|
| #ai_draft | AI 生成或整理过、尚未人工核对的内容 |
| #paper | 论文笔记或论文目录相关内容 |
| #tool | 可复用工具、定理或方法笔记 |

## 对象标签

| 标签                   | 含义                                               |
| -------------------- | ------------------------------------------------ |
| #hamiltonicity       | Hamiltonicity、Hamilton cycles、cycle space 等      |
| #expanders           | expanders、spectral expanders、C-expander 等        |
| #pseudorandom_graphs | pseudorandom graphs、random regular graphs 等      |
| #spanning_trees      | spanning trees、bounded-degree trees、hypertrees 等 |
| #hypergraphs         | hypergraphs 相关对象                                 |
| #dirac_type          | Dirac 条件、Dirac hypergraphs 等                     |
| #immersion_minor_subdivision | immersion、minor、subdivision、Hadwiger 型问题 |
| #graph_theory        | 一般图论概念，尽量少用作论文主标签                                |

## 合并规则

- Hamilton、Hamiltonicity、Hamilton-cycles 统一为 #hamiltonicity 。
- thresholds、spread、sunflower、fractional_expectation、Talagrand_conjecture 统一为 #spread_thresholds 。
- routing、edge_disjoint_paths、permutations、Hall_condition 统一为 #routing_connectors 。
- spectral_graph_theory、expander_mixing_lemma 等统一为 #spectral_method 。
- local_resilience、global_resilience、robustness 统一为 #resilience_perturbation 。
- absorb_method、distributive_absorption 统一为 #absorbing_method 。
- `未检查` 统一为 #ai_draft 。

## 使用原则

- 一篇论文笔记保留 3 到 6 个核心标签即可。
- 对象标签回答“研究什么”，方法标签回答“怎么证明”。
- 如果一个标签只会用一次，优先写进正文或关键词，不要放进 tags。
